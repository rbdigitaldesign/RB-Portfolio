Dual Coding Theory was developed by cognitive psychologist Allan Paivio in 1971. It proposes that the human mind operates with **two distinct but interconnected coding systems**: a verbal system (for language and text) and a non-verbal system (for images, spatial information, and mental imagery).

When information is encoded through *both* systems simultaneously, it creates two independent memory traces. This redundancy means:
- Recall has two retrieval pathways instead of one
- Understanding is deepened by the connections formed between verbal and visual representations
- Retention is significantly improved

**The two coding systems:**

**Verbal system:** Processes language — spoken or written words, sequential information, logical arguments. Information in this system is stored as *logogens* (word-like units).

**Non-verbal system:** Processes images, spatial relationships, and concrete objects. Information is stored as *imagens* (image-like units). This system is particularly powerful for representing spatial, dynamic, or concrete information.

**Three types of processing (in Mayer's extension):**
1. *Representational connections* — forming a mental representation from input (reading a word, seeing an image)
2. *Referential connections* — linking verbal and visual representations of the same concept
3. *Associative connections* — linking within the same system (word to word, image to image)

The richest learning activates all three, particularly referential connections between verbal and visual.

**Implications for instructional and interface design:**

- **Illustrated explanations outperform text-only explanations** for most types of procedural and conceptual learning
- **Animations with narration** are more effective than animations with on-screen text (because narration uses the verbal channel while the visual channel processes the animation — no channel conflict)
- **Images should be meaningful, not decorative** — decorative images add processing load without adding a second encoding pathway
- **Concept maps and diagrams** externalise verbal relationships visually, creating dual encoding
- **Worked examples with visual annotations** are more effective than text-only instructions

**Richard Mayer's multimedia learning principles (derived from dual coding):**
- *Coherence principle*: Remove extraneous material — it disrupts both channels
- *Signalling principle*: Use cues to direct attention in visuals
- *Contiguity principle*: Place related words and images near each other (temporal and spatial contiguity)
- *Modality principle*: Use narration rather than on-screen text when presenting alongside a visual

Dual coding theory is the reason "show, don't just tell" is not just a creative writing principle — it's a cognitive one.
