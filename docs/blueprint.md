# **App Name**: RB Digital Design Portfolio

## Core Features:

- Hero Section: Display a hero section with name, intro text, and circular profile image placeholder.
- Project Filtering: Enable filtering projects by category (UX, Learning Design, Hackathons, Coding Projects).
- Project Search: Provide a search bar for fuzzy matching on project titles/descriptions.
- Project Sorting: Sort projects by date (recent → oldest) and alphabetically (A–Z).
- Project Grid: Display projects in a responsive grid with lazy loading and accessible hover/focus states.
- Case Study Template: Use a case study template for detailed project views, including sections for overview, problem, process, and outcomes.
- Theme Toggle: Implement dark/light mode toggle with persisted user preference.

## Style Guidelines:

- Primary color: Dark green (#2F4F4F) for a professional and sophisticated look.
- Background color: Very light grey (#F5F5F2) for a clean, neutral backdrop.
- Accent color: Burnt orange (#FFB347) for highlighting key elements and calls to action, creating visual interest.
- Primary font: 'Inter' sans-serif for body text and headlines, providing a modern and readable experience. Note: currently only Google Fonts are supported.
- Secondary font: 'Merriweather' serif for a complementary, stylish appearance. Note: currently only Google Fonts are supported.
- Use clean, minimalist icons for navigation and category labels.
- Incorporate subtle animations for transitions and hover effects, respecting user preferences for reduced motion.
